Contact ConvertPDF

We're here to help! Whether you have questions, encountered a technical issue, want to suggest a new feature, or just want to say hello, we'd love to hear from you.

📧 Email Support

For general inquiries, support requests, or feedback, email us at:

convertpdf.contact@gmail.com

We aim to respond to all emails within 48 hours (and usually much sooner). Please include as much detail as possible—if you're reporting a bug, let us know which tool you were using, what browser and device you're on, and any steps to reproduce the issue.

🐛 Report a Bug

If you've found a bug or something isn't working as expected, the fastest way to get it fixed is to report it on our GitHub repository:

github.com/tony90lang/ConvertPDF/issues

When reporting a bug, please include:

  • Which tool you were using (e.g., Markdown to PDF, Merge PDFs, etc.)
  • Your browser and version (e.g., Chrome 122, Firefox 123, Safari 17)
  • Your operating system (Windows, macOS, Linux, Android, iOS)
  • Steps to reproduce the issue
  • Screenshots or screen recordings (if helpful)
  • Any error messages from the browser console (press F12 → Console tab)

💡 Feature Requests

Have an idea for a new tool or an improvement to an existing one? We'd love to hear it! You can:

  • Open a feature request on GitHub Issues with the label "enhancement"
  • Email us with the subject "Feature Request: [your idea]"

When suggesting a feature, please explain:

  • What problem it would solve
  • How you envision it working
  • Who would benefit from it
  • Any examples of similar tools (if applicable)

🤝 Contributing

ConvertPDF is an open-source project, and we welcome contributions from the community! Whether you're a developer, designer, writer, or just have good ideas, you can help make ConvertPDF better.

Check out our GitHub repository to get started. We have a contributor guide, code of conduct, and plenty of open issues tagged "good first issue" if you're looking for a place to start.

📢 Press and Partnerships

Are you a journalist, blogger, or content creator interested in covering ConvertPDF? Or do you have a partnership idea? Please email us with details, and we'll get back to you promptly.
